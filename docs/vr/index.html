VR (experimental)

Currently the player supports monoscopic (equirectangular) videos with 360-degree view, the original would look like this:

With the possibility to render them as a 360 panorama or in “cardboard mode” if the cardboard icon is clicked. To enable the VR feature showCardBoardView has to be set to true.

<video id='vr-video'>
    <source src='monoscopicvideo.mp4' title='1080p'  type="video/mp4"  />
</video>

<script>
fluidPlayer(
    'vr-video',
    {
        layoutControls: {
            showCardBoardView: true,
            showCardBoardJoystick: true
        }
    }
);
</script>


Ads

Currently we can show only 360degree video ads. Ads need to be served under VAST, with attribute mediaType=“360” in tag. Non-linear ads are not supported.
